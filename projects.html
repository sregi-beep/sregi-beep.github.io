Python · Machine Learning · Flask

Chess Engine

A Python chess engine combining classical alpha-beta search with a neural network evaluation function. The engine uses minimax with alpha-beta pruning (engine/search.py) alongside a custom neural network evaluator (engine/nn_model.py + nn_eval.py) trained to assess board positions beyond simple material counting. Deployed as a Flask web app on Render with a full browser-based board UI.

JavaScript · Quant Finance

Quant Persona App

A personal dashboard and daily quant practice tool. Features a 100+ problem bank spanning Probability, Statistics, Stochastic Calculus, Derivatives, Portfolio Theory, Linear Algebra, Risk, and Finance — organized across 5 difficulty levels (Intro → Expert). A seeded daily shuffle delivers 10 problems per day, with topic filters, a show/hide answer flow, and a live markets dashboard and news feed. Built entirely in vanilla JS with no dependencies.
